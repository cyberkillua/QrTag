# qr-code

Helps you generate QR codes for your favourite websites 

## Project setup
```
yarn install
```
### Technologies
```
Vue, VueX and Vuetify
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
